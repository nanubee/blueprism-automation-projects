# Currency Converter Automation

## Overview

A Blue Prism RPA solution that automates currency conversion using a web-based currency calculator. The automation reads currency details from an Excel file, performs the conversion through the website, retrieves the converted amount, and writes the result back to the Excel sheet.

---

## Features

- Launches a web browser
- Navigates to the currency converter website
- Reads input data from Excel
- Enters source currency
- Enters target currency
- Enters conversion amount
- Retrieves converted value
- Updates the Excel file with the result

---

## Repository Structure

### BPA Object - Currency_Web_Actions.bpobject

Contains reusable actions responsible for interacting with the currency conversion website, including browser navigation, entering values, reading results, and updating Excel.

### BPA Process - Currency_Dispatcher.bpprocess

Reads input records from the Excel file and places them into a Blue Prism Work Queue for processing.

### BPA Process - Currency_Performer.bpprocess

Processes each queue item by launching the browser, performing the currency conversion, retrieving the converted value, and updating the output.

### Currency_Data.xlsx

Sample input and output workbook used by the automation. It contains the currencies, amounts to convert, and the converted values generated by the bot.
