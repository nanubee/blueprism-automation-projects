# Currency Converter Automation

## Overview

A Blue Prism automation that performs currency conversion through a web-based currency calculator and update in excel sheet.

## Features

- Launch browser
- Navigate to website
- Enter source currency
- Enter target currency
- Enter amount
- Retrieve converted value

## Technologies

- Blue Prism 7.3
- Browser Automation

## Learning Outcomes

- Application Modeller
- Browser spying
- Navigate stages
- Read and Write stages
