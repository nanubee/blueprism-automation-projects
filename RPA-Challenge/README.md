# RPA Challenge Automation

## Overview

A Blue Prism automation that completes the RPA Challenge by reading input data and populating dynamic web forms. The solution demonstrates browser automation, dynamic web element identification, and reliable data-driven processing.

## Challenge

This project automates the **RPA Challenge**, a widely used benchmarking exercise for Robotic Process Automation tools that tests an automation's ability to handle dynamic web elements and data-driven form completion.

https://rpachallenge.com/

---

## Features

- Launches and navigates the browser
- Reads input data from the RPA Challenge website
- Identifies dynamic web elements
- Automatically populates web forms
- Submits data accurately
- Handles dynamic selectors during execution
