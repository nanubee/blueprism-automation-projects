# RPA Challenge Automation

## Overview

This project automates the RPA Challenge website using Blue Prism.

The bot reads input data and populates dynamic web forms while handling changing web element attributes.

## Features

- Dynamic element identification
- Data collection processing
- Browser automation
- Exception handling

## Technologies

- Blue Prism 7.3

## Learning Outcomes

- Browser automation
- Collections
- Dynamic selectors
- Process design
