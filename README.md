# Blue Prism Automation Projects

A collection of Robotic Process Automation (RPA) projects developed using **Blue Prism 7.3**.

## Projects

### 1. RPA Challenge Automation

Automates the RPA Challenge web application by reading input data and completing dynamic web forms.

**Features**

- Browser automation
- Dynamic web element interaction
- Data-driven automation
- Collection handling
- Exception handling

---

### 2. Currency Converter Automation

Automates the currency conversion process using an online currency converter.

**Features**

- Launches browser
- Navigates to the currency converter website
- Inputs currencies and amount
- Reads converted value
- Browser automation

---

## Technologies

- Blue Prism 7.3
- Browser Automation
- Excel Integration
- Collections
- Exception Handling
